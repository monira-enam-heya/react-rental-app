# React Car Rental App
This app is created with react.js for renting cars. Data stored in json file

# What is used:
Npx
Jsx
Bootstrap

# Version Requirement:
Node >= 14.0.0 and npm >= 5.6 

# Instalation Instruction
1. Clone the repo
2. Run "npm run build"
3. Project will run on http://localhost:3000
